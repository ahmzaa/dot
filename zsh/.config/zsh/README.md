## alias usage

### list
- `ls` - `ls -1 --color`
- `ll` - `ls -lh --color`
- `la` - `ls -alh --color`

### neovim
- `vim` - `nvim`
- `vi` - `nvim`

### git (gx)
- `ga` - `git add`
- `gp` - `gitpush()` - git pull & git push
- `gs` - `git status`
- `gc` - `git commit`

### docker
- `dc` - `docker compose`
- `dcu` - `docker compose up`
- `dcd` - `docker compose down`

### dotfiles
- `dotconf` - `/usr/bin/git --git-dir=$HOME/.dot --work-tree=$HOME`
